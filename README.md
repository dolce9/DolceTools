# DolceTools for Unity

VRChat 用 Unity 拡張です。初めて作った自分用ツールのため、導入には十分注意してください。

---

## ✨ 主な機能

### Project ウィンドウ拡張
- ファイルの拡張子を表示
- フォルダ内のファイル数・フォルダ数を表示

### Hierarchy 拡張
- コンポーネントのアイコン表示
- EditorOnly オブジェクトの背景色表示
- EditorOnly のワンクリック切り替え
- Active のワンクリック切り替え

### コンポーネント
- **DLTComment**：ビルド時に削除されるメモ（NDMFに依存）

---

## 📦 インストール方法（VPM / VCC）

1. 次のページから「VCC に追加」をクリック：https://dolce9.github.io/vpm/
2. プロジェクトの「Manage Project」から **DolceTools for Unity** をインストールしてください。

---

## 🔧 使い方

### メニュー場所
Unity 上部メニュー `Window → DolceTools` もしくは `Tools → DolceTools → Settings（設定）` から各機能の ON / OFF を切り替え可能です。
